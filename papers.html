Corea Starstroupe Research Paper Archive

The Corea Starstroupe Research Paper Archive, a curated collection of our open-source contributions in natural language processing, embedded AI, and human-computer interaction.

  • Intentional Systems: Designing Computational Frameworks for Human Purpose Alignment
    Ron Asnahon, June 2023
    Explores computational frameworks designed to align AI systems with human intentions, focusing on ethical and purposeful interaction models.
  • Nexora Proto v0.1: Foundational Build of a Compact NLP Stack for Resource-Constrained Devices
    Ron Asnahon, November 2023
    Details the initial development of Nexora Proto, a sub-million-parameter NLP model optimized for microcontrollers, with focus on tokenization and embedding stability.
  • NeuroLite-4M: A Compact Small Language Model for Real-Time Edge Processing
    Ron Asnahon, December 2023
    Introduces NeuroLite-4M, a compact language model designed for real-time NLP tasks on edge devices, emphasizing low-latency inference.
  • Semantic Presence: A Framework for Emotionally and Spatially Aware Human-Computer Interaction
    Ron Asnahon, February 2024
    Proposes a framework for enhancing human-computer interaction through emotionally and spatially aware interfaces, leveraging semantic context.
  • PEACHI OS: Foundational Implementation for Natural Language Processing Workloads
    Ron Asnahon, March 2024
    Outlines the foundational implementation of PEACHI OS, a lightweight operating system for NLP workloads, with focus on microkernel modularity and token scheduling.
  • Nexora NeuroLite v0.6.2: Stabilization and Optimization for Embedded NLP Platforms
    Ron Asnahon, April 2024
    Presents the stabilization phase of Nexora NeuroLite v0.6.2, with enhancements in training compression, scheduling, and vector integrity for embedded NLP.
  • Nexora Neurolite v1.0: Architectural Evolution for Lightweight NLP on Embedded Devices
    Ron Asnahon, May 2024
    Chronicles the evolution from Nexora Proto to Neurolite v1.0, a production-ready model for embedded NLP with improved compression and instruction tuning.
  • Auralis v0.1: A Compact NLP Model for Lightweight Instruction-Following and Real-Time Comprehension
    Ron Asnahon, July 2024
    Documents the initial development of Auralis v0.1, a compact NLP model for instruction-following and dialog intent extraction, with focus on interpretability.
  • Intent Vector System: A Computational Framework for Real-Time User Intent Modeling in Peachi OS
    Ron Asnahon, August 2024
    Introduces a computational framework for real-time user intent modeling within PEACHI OS, enhancing conversational intelligence.
  • PEACHI OS: Adaptive Kernel Computation for Conversational Intelligence
    Ron Asnahon, October 2024
    Details advancements in PEACHI OS’s adaptive kernel for conversational AI, focusing on dynamic computation for real-time dialogue.
  • Adaptive Cognitive Interfaces: The Lumen Layer Framework in Project Mindmesh
    Ron Asnahon, December 2024
    Explores the Lumen Layer framework in Project Mindmesh, enabling adaptive cognitive interfaces for enhanced user interaction.
  • Auralis: A Next-Generation Small Language Model for Nuanced NLP in Resource-Constrained Environments
    Ron Asnahon, January 2025
    Presents Auralis as a next-generation NLP model for nuanced tasks in low-resource settings, building on prior Auralis developments.
  • Project Mindmesh: The Lumen Layer – A Perceptual Operating System Architecture
    Ron Asnahon, May 2025
    Details the Lumen Layer in Project Mindmesh, a perceptual OS architecture for advanced human-computer interaction.
